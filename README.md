# Bagas-s-Journal (Silent Script)

Aplikasi Voice Journal yang memungkinkan pengguna untuk berbicara, dan secara otomatis dirapikan oleh AI, lalu ditampilkan dalam format tulisan tangan di atas kertas bergaris virtual.

## Fitur
- **Speech-to-Text**: Mengubah suara menjadi teks menggunakan Web Speech API.
- **AI Refiner**: Merapikan teks (mock-up/placeholder).
- **Handwriting Visuals**: Merender teks menggunakan font bergaya tulisan tangan.
- **Save Image**: Menyimpan catatan sebagai file gambar PNG.
- **Auto-Save**: Menyimpan catatan sementara ke `localStorage`.

## Menjalankan Aplikasi Lokal
1. `npm install`
2. `npm run dev`

## Deployment
